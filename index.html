About

This site was invented to quickly tag people from an Instagram post - in the description with links for YouTube.

The site works with Instagram, YouTube, TikTok and Telegram type link

Example

Instagram post:

Hana as Niki @kamilla.rst
Tori as Jugwon @msvxl

After convertor with config "Instagram"

Hana as Niki https://www.instagram.com/kamilla.rst
Tori as Jugwon https://www.instagram.com/msvxl

After convertor with config "Instagram" and "Only Links"

https://www.instagram.com/kamilla.rst
https://www.instagram.com/msvxl
